# Projeto AspNet API para Controle de Usuários
 Essa Api já existe em meu repositório, porém o diferencial dela que a chave secreta
 de antifalsificação do token eu preferi trocar para colocar na FornecedorApi.

## Tecnologias utilizadas:
* AspNet 7 API
* Microserviços REST
* Token de autenticação JWT
* Swagger
* Entity Framework Core e acesso ao SqlServer(criar banco com o nome BD_ApiUsuarios)
* DDD(Domain-Driven Design)
* TDD(Test Driven Development)
